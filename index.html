This application ("App") is developed to help users save and manage parking locations, including optional photo/video records and backup to Google Drive.

1. Information We Collect

The App may collect the following information:

  • Location data (to save parking positions)
  • Photos or videos (if user chooses to add parking records)
  • Device storage access (to save or load media)

We do NOT collect personal identity information such as name, email, or phone number, unless required for Google Drive integration.

2. How We Use Information

  • Saving parking locations
  • Displaying maps and navigation
  • Storing parking photos/videos
  • Backup and restore via Google Drive

3. Data Storage

All data is stored locally on the user's device. If Google Drive backup is used, files are stored in the user’s own Google Drive account. We do NOT store user data on external servers.

4. Camera Permission

The app uses camera access only to capture parking photos. No data is collected or shared without user action.

5. Permissions

  • Location → to save parking position
  • Camera → to take parking photos
  • Storage → to save and load media

6. Data Sharing

We do NOT sell, trade, or share user data with third parties.

7. Security

We take reasonable measures to protect user data, but no method is 100% secure.

8. User Control

  • Users can delete data anytime from the app
  • Users can disable permissions via device settings
  • Users can stop using backup features anytime

9. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be reflected on this page.
